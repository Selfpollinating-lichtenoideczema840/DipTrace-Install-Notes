## ⚠️ Troubleshooting Installation Errors
If the installer fails, follow these methods to fix the issue:

### Check Compatibility Settings
Sometimes Windows 11 blocks older installers. 
1. Right-click the installation file.
2. Select "Properties".
3. Navigate to the "Compatibility" tab.
4. Check the box labeled "Run this program in compatibility mode for".
5. Choose "Windows 8" or "Windows 7" from the dropdown menu.
6. Check "Run this program as an administrator".
7. Click "Apply" and then "OK".
8. Run the installer again.

### Disable Antivirus Temporarily
Antivirus software may flag legitimate installers as a risk. 
1. Open your Windows Security settings.
2. Go to "Virus & threat protection".
3. Select "Manage settings".
4. Turn off "Real-time protection" for a few minutes.
5. Attempt the installation again.
6. Turn the protection back on once the installation finishes.

### Verify User Permissions
Ensure your user account has administrative privileges. If you share a computer, log in to the primary account to install shared software. In some cases, Windows 11 User Account Control (UAC) prevents silent installs. Lowering your UAC settings temporarily via the Control Panel can resolve hidden permission conflicts.
